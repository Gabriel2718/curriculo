Páginas web

Projeto Front-End - Saúde Paraná

Página estática desenvolvida com HTML e CSS para a disciplina de Desenvolvimento Front-End da graduação em Análise e Desenvolvimento de Sistemas. A página visa apresentar informações sobre algumas das doenças endêmicas mais comuns Paraná.

Página React - At Full Steam

Página Front-End desenvolvida utilizando React, que utiliza uma API externa para obter dados de jogos em promoção na plataforma Steam e exibi-los em um grid de vizualização.

Aplicativos móveis

Aplicativo Flutter - Coinverter

Aplicativo para Android, desenvolvido com Flutter, que realiza conversões entre moedas (Real, Dólar, Euro e Bitcoin) utilizando uma API (AwesomeAPI) para obter os valores atualizados de cada moeda.

Aplicativo Flutter - GIFinder

Aplicativo para Android, desenvolvido com Flutter, que permite ao usuário visualizar, pesquisar (por meio de filtros inseridos na requisição) e compartilhar GIFs obtidos da API do site GIPHY.

Aplicativo Flutter - Agenda de Contatos

Aplicativo para Android, desenvolvido com Flutter, no qual o usuário pode cadastrar informações de um contato (nome, e-mail, telefone e uma foto), bem como editá-las e removê-las.
